# ゲームテンポ・ゲーム性
-------------------------------------------------------------------------
 - [x] ゆったり / カジュアル （落ち着いて遊べる、時間に余裕がある、難易度上昇が緩やか）
 - [ ] 激しい / ハードコア （厳しい当たり判定、失敗と再挑戦を繰り返す高難易度）

## 進行形式
 - [x] エンドレス （終了まで続く無限スコアアタック）
  - [ ] ステージ制 （ステージを順番にクリアして進む）
 - [ ] ストーリー進行 （物語に沿ってゲームが進む）

## 難易度カーブ
 - [x] 緩やか （時間や進行に応じて少しずつ難しくなる）
 - [ ] 動的調整 （プレイヤーの成績に応じて難易度を調整する）
 - [ ] 固定 （最初から最後まで一定の難易度）

## プレイヤー体験
 - [x] 覚えやすい （ルールが単純で、直感的に操作できる）
 - [ ] 奥深い上達要素 （上達するほど高度なプレイが可能になる）
 - [ ] 精密操作重視 （正確な移動やタイミングが必要）

## 失敗時のペナルティ
 - [x] やさしい （失敗しても立て直しやすい）
 - [ ] 厳しい （失敗が大きな不利益やゲームオーバーにつながる）

## 1回のプレイ時間
 - [x] 短時間 （1〜5分）
 - [ ] 中程度 （5〜15分）
 - [ ] 長時間 （15分以上）

## 繰り返し遊べる要素
 - [x] 高いリプレイ性 （スコアアタック、エンドレス、ランダム要素など）
 - [ ] 1回の体験を重視 （物語や演出を中心とした、一度通して遊ぶ内容）

## コンテンツ生成
 - [x] ランダム生成
 - [ ] 固定配置
